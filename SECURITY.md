# Security Policy

## Supported Versions

Security updates are provided for the latest major release.

| Version | Supported |
| :--- | :---: |
| 2.0.x | ✅ |
| 1.0.x | ❌ |

## Reporting a Vulnerability

If you discover a potential security issue in Weatherpulse, please notify us immediately. We will investigate all legitimate reports.

Please report security issues directly by opening an issue on GitHub.
